PHP-QBO
==========================
A PHP library to access Quickbooks online / QBO.

- PSR compatible


Requirements
------------
- PHP >= 5.6


Install
-------

### Composer

Add a dependency on `beethoven/phpqbo` to your project's `composer.json` file.

```javascript
{
    "require": {
        "beethoven/phpqbo": "0.0.1-alpha"
    }
}
```

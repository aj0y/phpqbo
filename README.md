# phpqbo
PHP library to access Quickbooks online (a.k.a QBO)

PHP-QBO
==========================
A PHP library to access Quickbooks online / QBO.


Requirements
------------
- PHP >= 5.6


Install
-------

### Composer

Add a dependency on `aj0y/phpqbo` to your project's `composer.json` file.

```javascript
{
    "require": {
        "aj0y/phpqbo": "1.*"
    }
}
```
